Программа «Помощник вертикальца»
1.Возможности программы: 
1) построение графиков функций; 
2) решение квадратных уравнений; 
3) разложение квадратного трёхчлена на множители; 
4) выяснение синусов, косинусов, тангенсов и котангенсов для разных углов; 
5) просто калькулятор для счёта;
6) проверка знаний ученика на эту тему(к каждой теме будут прикреплены тесты для проверки знаний на эту тему);
7) теоретический материал для каждой темы.
2. Для чего она создана?
Для помощи и объяснения школьникам некоторых тем алгебры и геометрии.
